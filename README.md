# 时之堡 - Time Castle

**版本**: 1.0  
**引擎**: Ren'Py 8.5.2  
**类型**: 时间旅行视觉小说  
**时长**: 2-3 小时  

---

## 📋 游戏内容

### 角色
- **我** - 主角，时间旅行者
- **艾登** - 时之堡守护者，引导玩家
- **旁白** - 叙述故事

### 章节结构
- **第一章**: 事故 - 遇见艾登，了解任务
- **第二章**: 四时代 - 穿越到中世纪/法国大革命/二战
- **第三章**: 线索 - 收集时间碎片
- **第四章**: 抉择 - 改变历史 or 保持原貌
- **第五章**: 时间锚点 - 结局

### 分支系统
- **3 个时代选择**: 中世纪 (1347)、法国大革命 (1789)、二战 (1943)
- **多对话分支**: 每个场景 2-3 个选择
- **多结局**: 改变者/守护者/时间旅行者

### 时代背景
1. **中世纪 (1347 年)** - 黑死病肆虐的欧洲
2. **法国大革命 (1789 年)** - 巴黎革命浪潮
3. **二战时期 (1943 年)** - 战火纷飞的欧洲

---

## 🎮 快速开始

1. 打开 Ren'Py Launcher
2. 选择「TimeCastle_1」
3. 点击「启动」
